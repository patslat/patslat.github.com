Create

My first project was a program for an fMRI experiment examining the effects of mood on cognition in schizophrenia patients. I've since completed and deployed several projects which can be found here:

aNimal back
A Rails and Backbone.js implementation of the Dual n-back, a task that is commonly used to measure working memory in cognitive neuroscience experiments. Optional authentication for stat-tracking uses Devise with Omniauth integration for easy Facebook log-in. User performance data is graphed using D3.js.


Droptask Clone
A Rails and Backbone.js clone of DropTask, a visual task-managment application. Uses Backbone-relational to model nested resources. Uses jQuery UI to allow users to interact with tasks by dragging and dropping them.


Happ
A Rails and Backbone.js personal tracking app. Users provide daily 0-10 ratings on five aspects of their day. Uses D3.js to display graphs of single days and a summary of all data. Combines Kaminari and AJAX to make entries viewable by infinite-scroll.
